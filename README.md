# fibo-practice
Fibonachi Sequence with Recusion
f(n) = \begin{cases}
0 & \text{if } n = 0 \\
1 & \text{if } n = 1 \\
F(n-1) + F(n-2) & \text{if } n > 1
\end{cases}


## Prerequisites
- Python ^3.12
- Pandas =1.0.0

## How to Start
``` shell
$ git clone https://github.com/kimmary96/fibo-practice.git
$ cd fibo-practice
$ python fibo-first.py
```
## Installation
`$pip install -r requirements.txt`
